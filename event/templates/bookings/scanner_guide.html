QR Scanner User Guide

Complete step-by-step guide to scanning and verifying tickets

Quick Start

  1. Login as staff member
  2. Click "QR Scanner" in your profile menu
  3. Allow camera access when prompted
  4. Click "Start Scanner" button
  5. Point camera at ticket QR code
  6. Review result and admit if valid
Open Scanner Now

Detailed Steps

1Access the Scanner

Navigate to the QR Scanner from your profile dropdown menu (staff only).

Direct URL: http://127.0.0.1:8000/bookings/scanner/

2Grant Camera Permissions

When prompted, click "Allow" to grant camera access to your browser.

Desktop Browser
Click "Allow" in the popup. If blocked, click the camera icon in the address bar.
Mobile Device
Tap "Allow" in the system prompt. Check Settings > Browser > Permissions if denied.

3Start Scanning

Click the "Start Scanner" button and wait for the camera to initialize (2-3 seconds).

  • Camera feed will appear
  • Scanning frame visible in center
  • Button changes to "Stop Scanner"

4Scan QR Codes

Point your camera at the QR code on the ticket:

  • Center the QR code in the frame
  • Keep distance of 10-30cm (4-12 inches)
  • Ensure good lighting
  • Hold device steady
  • Scanner automatically detects and reads

5Review Results

After scanning, a result card appears showing ticket status:

Valid Ticket
Green card - Click "Admit Attendee" to allow entry
Already Used
Orange card - Ticket already scanned, deny entry
Invalid Ticket
Red card - Ticket not found or cancelled, deny entry

Troubleshooting

  • Check browser permissions in settings
  • Try a different browser (Chrome recommended)
  • Restart the browser
  • Ensure no other app is using the camera
  • Clean the camera lens

  • Ensure QR code is fully visible in frame
  • Improve lighting (avoid glare and shadows)
  • Hold device steady
  • Try different angles and distances
  • Check if QR code is damaged or distorted
  • Increase screen brightness if scanning from phone

  • Check internet connection
  • Refresh the page and try again
  • Verify ticket in admin panel manually
  • Contact support if issue persists

Best Practices

Do's
  • Test scanner before event starts
  • Keep device charged
  • Position in well-lit area
  • Verify attendee ID for security
  • Keep scan history for reference
  • Report suspicious tickets
Don'ts
  • Don't share scanner access
  • Don't scan too quickly
  • Don't admit invalid tickets
  • Don't forget to log out
  • Don't ignore network issues
  • Don't leave device unattended
